# PH-4502C pH Sensor

The **PH-4502C** module was used in the project to measure the hydrogen potential (pH) of the water. Measuring the pH makes it possible to determine whether the water is acidic, neutral, or alkaline, making it an important indicator of the water quality supplied to the drinking trough.

![pH Scale](/docs/Hardware/Sensores/img/pH.png)

---

# How It Works

The sensor consists of a **glass electrode probe** connected to the PH-4502C module through a BNC connector.

When the probe is immersed in a solution, it measures the concentration of hydrogen ions (H⁺). This measurement generates a very small electrical signal (on the order of millivolts), which is amplified by the PH-4502C module and made available through the **PO** analog output, allowing it to be read by the ESP32 ADC.

# Pinout

The module provides six pins.

| Pin          | Function                                                              | ESP32 Connection |
| ------------ | --------------------------------------------------------------------- | ---------------- |
| TO           | Temperature sensor output                                             | Not used         |
| DO           | Digital output (threshold defined by the pH adjustment potentiometer) | Optional         |
| PO           | Analog pH sensor output                                               | GPIO34 (ADC)     |
| GND (Probe)  | Probe ground                                                          | GND              |
| GND (Module) | Module ground                                                         | GND              |
| VCC          | Power supply (5 V)                                                    | Vin (5 V)        |
| POT1         | Analog output offset adjustment                                       | —                |
| POT2         | Digital output threshold adjustment                                   | —                |

Only three pins were used in this project:

* **V+** → Vin (5 V)
* **PO** → GPIO34
* **G** → GND

# Test Code

```cpp
#include <Arduino.h>

// Pin definition
#define PH_PIN 34

// Reference voltage measured in neutral water (pH 7)
float tensao_ph7 = 2.688;

// Voltage-to-pH conversion slope
float slope = -120.0;

// Buffer used to filter ADC readings
int buffer_arr[10], temp;

void setup() {
  Serial.begin(115200);
  delay(1000);

  analogReadResolution(12);
  analogSetPinAttenuation(PH_PIN, ADC_11db);
}

int lerADCFiltrado() {

  for (int i = 0; i < 10; i++) {
    buffer_arr[i] = analogRead(PH_PIN);
    delay(30);
  }

  for (int i = 0; i < 9; i++) {
    for (int j = i + 1; j < 10; j++) {
      if (buffer_arr[i] > buffer_arr[j]) {
        temp = buffer_arr[i];
        buffer_arr[i] = buffer_arr[j];
        buffer_arr[j] = temp;
      }
    }
  }

  long soma = 0;

  for (int i = 2; i < 8; i++)
    soma += buffer_arr[i];

  return soma / 6;
}

void loop() {

  int adc = lerADCFiltrado();

  float tensao = adc * (3.3 / 4095.0);

  float ph = 7 + slope * (tensao - tensao_ph7);

  if (ph < 0) ph = 0;
  if (ph > 14) ph = 14;

  Serial.print("Voltage: ");
  Serial.print(tensao,3);
  Serial.print(" V | pH: ");
  Serial.println(ph,2);

  delay(500);
}
```

# Calibration

The pH probe generates a small electrical voltage that varies according to the pH of the solution.

| pH | Probe Voltage |
| -- | ------------- |
| 0  | +414 mV       |
| 7  | 0 mV          |
| 14 | -414 mV       |

Since most ADCs cannot measure negative voltages, the PH-4502C module adds an **offset** to its analog output.

This offset is adjusted using **POT1**, located next to the BNC connector.

## Offset Adjustment

1. Disconnect the probe from the BNC connector.
2. Short-circuit the BNC connector (center pin connected to the outer shield). This simulates a reading of approximately **0 mV**, corresponding to **pH 7**.
3. Measure the voltage available at the **PO** output.
4. Adjust **POT1** until the output voltage reaches approximately **2.5 V**.

> **Figure:** Short-circuiting the BNC connector.

![BNC Short Circuit](/docs/Hardware/Sensores/img/img7.png)

After this procedure, the voltage curve becomes centered, allowing the ESP32 to correctly measure both acidic and alkaline solutions.
