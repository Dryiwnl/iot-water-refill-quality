# Sensor de pH PH-4502C

O módulo **PH-4502C** foi utilizado no projeto para medir o potencial hidrogeniônico (pH) da água. A medição do pH permite determinar se a água está ácida, neutra ou básica, sendo um importante indicador da qualidade da água utilizada no bebedouro.

![Escala de pH](/docs/Hardware/Sensores/img/pH.png)

---

# Como funciona

O sensor é composto por uma **sonda de vidro** conectada ao módulo PH-4502C através de um conector BNC.

Quando a sonda é imersa em uma solução, ela mede a concentração de íons de hidrogênio (H⁺). Essa medição gera um sinal elétrico extremamente pequeno (na ordem de milivolts), que é amplificado pelo módulo PH-4502C e disponibilizado na saída analógica **PO**, permitindo sua leitura pelo ADC do ESP32.

# Pinagem

O módulo possui seis pinos.

| Pino         | Função                                                      | Conexão no ESP32 |
| ------------ | ----------------------------------------------------------- | ---------------- |
| TO           | Saída do sensor de temperatura                              | Não utilizado    |
| DO           | Saída digital (definida pelo potenciômetro de limite de pH) | Opcional         |
| PO           | Saída analógica do sensor de pH                             | GPIO34 (ADC)     |
| GND (Sonda)  | Terra da sonda                                              | GND              |
| GND (Módulo) | Terra do módulo                                             | GND              |
| VCC          | Alimentação (5 V)                                           | Vin (5 V)        |
| POT1         | Ajuste do offset da saída analógica                         | —                |
| POT2         | Ajuste do limite da saída digital                           | —                |

No projeto foram utilizados apenas três pinos:

* **V+** → Vin (5 V)
* **PO** → GPIO34
* **G** → GND

# Código de teste

```cpp
#include <Arduino.h>

// Definicao da pinagem
#define PH_PIN 34

// Tensao de referencia medida em agua neutra (pH 7)
float tensao_ph7 = 2.688;

// Ganho da conversao de tensao para pH
float slope = -120.0;

// Vetor auxiliar para filtrar as leituras
int buffer_arr[10], temp;

void setup() {
  Serial.begin(115200);
  delay(1000);

  analogReadResolution(12);
  analogSetPinAttenuation(PH_PIN, ADC_11db);
}

int lerADCFiltrado() {

  for (int i = 0; i < 10; i++) {
    buffer_arr[i] = analogRead(PH_PIN);
    delay(30);
  }

  for (int i = 0; i < 9; i++) {
    for (int j = i + 1; j < 10; j++) {
      if (buffer_arr[i] > buffer_arr[j]) {
        temp = buffer_arr[i];
        buffer_arr[i] = buffer_arr[j];
        buffer_arr[j] = temp;
      }
    }
  }

  long soma = 0;

  for (int i = 2; i < 8; i++)
    soma += buffer_arr[i];

  return soma / 6;
}

void loop() {

  int adc = lerADCFiltrado();

  float tensao = adc * (3.3 / 4095.0);

  float ph = 7 + slope * (tensao - tensao_ph7);

  if (ph < 0) ph = 0;
  if (ph > 14) ph = 14;

  Serial.print("Tensao_pH:");
  Serial.print(tensao,3);
  Serial.print("V | pH:");
  Serial.println(ph,2);

  delay(500);
}
```

# Calibração

A sonda de pH gera uma pequena tensão elétrica que varia conforme o pH da solução.

| pH | Tensão da sonda |
| -- | --------------- |
| 0  | +414 mV         |
| 7  | 0 mV            |
| 14 | -414 mV         |

Como a maioria dos conversores ADC não consegue medir tensões negativas, o módulo PH-4502C adiciona um **offset** à saída analógica.

Esse deslocamento é ajustado pelo **POT1**, localizado próximo ao conector BNC.

## Ajuste do offset

1. Remova a sonda do conector BNC.
2. Curto-circuite o conector BNC (pino central com a carcaça externa). Esse procedimento simula uma leitura de aproximadamente **0 mV**, correspondente ao **pH 7**.
3. Meça a tensão presente na saída **PO**.
4. Ajuste o **POT1** até obter aproximadamente **2,5 V**.

> **Imagem:** Curto-circuito do conector BNC.

![Curto do BNC](/docs/Hardware/Sensores/img/img7.png)

Após esse procedimento, a curva de tensão ficará centralizada, permitindo que o ESP32 leia corretamente tanto soluções ácidas quanto básicas.
